# PrimaryStock

## 概要

長期投資に資する日本株を検索するツール

## 実行方法

開発者向け(Python):
python PrimaryStock.py

利用者向け(EXE):
PrimaryStock.exeを実行

## 出力先

output/YYYYMMDD_HHMMSS/

## 推奨環境

windows11
Python 3.12+

## インストール (Python)

```bash
git clone https://github.com/Tama-1120/PrimaryStock.git
pip install -r requirements.txt
